# AI Tutor Prompt Generator

A bilingual web page that helps children (ages 10-12) create personalized AI tutor prompts for safe use in ChatGPT or Google Gemini.

## How to Use

1. Open `https://teal-projects.github.io/build-your-own-ai-tutor/` in a web browser
2. View the interactive demo showing AI response differences
3. Fill out the form with student information
4. Click "Create Prompt" to generate personalized tutor prompt
5. Copy the prompt and paste it into ChatGPT or Gemini
6. Start asking homework questions

## Features

- **Bilingual interface** - Switch between German and English
- **Interactive demo** - See the difference between regular AI and tutor responses
- **Personalized prompts** - Fill in grade, languages, subjects, and learning preferences  
- **One-click copy** - Copy generated prompt to clipboard
- **Progress tracking** - Visual progress bar (Fill ‚Üí Copy ‚Üí Use)
- **Local storage** - Saves form data between sessions
- **Safe usage instructions** - Age-appropriate guidance for AI chat tools

## Safety Features

- Client-side only - no data sent to servers
- Educational guidance prompts only
- Redirects inappropriate questions to parents/adults
- Age-appropriate language and instructions
